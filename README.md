# GreenDev Associates — Corporate Website

Official website for **GreenDev Associates**, an environmental consulting firm.

🌐 **Live Site:** [greendevassociates.com](https://greendevassociates.com)

---

## About the Project

GreenDev Associates needed a professional corporate website to establish their digital presence and communicate their environmental consulting services to clients. The site was designed and developed as a clean, modern single-page React application.

### Key Features

- **Company Overview** — Professional presentation of the firm's mission and services
- **Services Section** — Clear breakdown of environmental consulting offerings
- **Contact Integration** — Easy-to-use contact section for client enquiries
- **Responsive Design** — Fully optimised across all screen sizes
- **Fast Performance** — Lightweight React build with optimised assets
